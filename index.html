Install

This is your “install page” inside the site—step‑by‑step instructions plus a dedicated download card.

  1. 1
    Install NeoForge 1.21.8
    Download and install the NeoForge 1.21.8 loader for your Minecraft installation. Make sure you can select the NeoForge profile in your launcher.
  2. 2
    Download KullTools 3.1.0 Beta
    Go to the GitHub Releases page and download the file: kulltools-3.1.0 Beta-neoforge-1.21.8.jar.
  3. 3
    Place the JAR in your mods folder
    Open your Minecraft directory and drop the JAR into the mods folder for the NeoForge 1.21.8 instance.
  4. 4
    Launch Minecraft with NeoForge
    Start the game using the NeoForge 1.21.8 profile. KullTools should appear in the Mods list with your custom icon.
  5. 5
    Enter the Kull dimension
    Use your mod’s progression (portal, item, or structure) to access the new fiery biome and start exploring.
Note: Always back up your worlds before adding new mods, especially beta versions. KullTools 3.1.0 is a beta release and may change or break between updates.

FAQ

You can edit these answers to match your mod’s exact behavior and requirements.

What Minecraft version is supported?

KullTools 3.1.0 Beta is built for Minecraft NeoForge 1.21.8. Using it on other versions is not recommended and may cause crashes or unexpected behavior.

Is this mod safe for existing worlds?

It is always safer to use a new world when testing beta mods. If you add KullTools to an existing world, make a backup first.

Can I use this in a modpack?

Yes, as long as you follow the license and credit the project. Link back to the GitHub repository and this page so players can find updates.

Where do I report bugs or suggest features?

Use the Issues tab on the GitHub repository to report bugs or request new features. Include logs, screenshots, and steps to reproduce when possible.
